AS PLAY - Java Music Management System 🎵

A robust, console-based Music Playlist Management System built in Java. This project demonstrates the implementation of core Data Structures (Linked Lists, Stacks, Queues, and Arrays) from scratch to manage a music library, track listening history, and prioritize upcoming songs.

🚀 Features

. Playlist Management (Linked List): Add, delete, and display songs using a dynamic singly linked list.
. Top 5 Leaderboard (Arrays): Automatically tracks and sorts the top 5 highest-rated songs in a fixed-size array.
. Recently Played (Manual Stack): Implements a Last-In, First-Out (LIFO) history tracker to see your most recent tracks.
. Play Next Queue (Manual Queue): Implements a First-In, First-Out (FIFO) system to manage a "waiting list" of songs.
. Audio Playback: Native support for .wav file playback using the javax.sound.sampled library.
. Persistent Logging: Automatically saves the current state of the playlist, history, and top songs to a college_playlist.txt file after every action.

🛠️ Technical Implementation

. Unlike standard applications that use java.util.Collections, this project focuses on the manual manipulation of nodes and pointers to prove a deep understanding of memory management:
. Custom Nodes: Defined a DataNode class to handle stack and queue operations manually.
. Pointer Logic: Managed head, Top, Front, and Rear pointers to control data flow without using .push(), .pop(), or .add() from the standard library.
. Search & Sort: Implements linear search for song lookups and a shift-based sorting algorithm for the Top 5 array.
